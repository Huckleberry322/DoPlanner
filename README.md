# DoPlanner

### Description
Test compose app. Full Kotlin.

### Tech stack:
Compose (ver 1.1.1), MVVM, Hilt, DataStore, Json.

### Compose features:
Compose navigation (with Hilt), Accompanist UI controller, Compose LiveData, Compose Material.
